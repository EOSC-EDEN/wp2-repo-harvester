
# Contributing

## Branch Policy

The `master` branch is the default branch

### How to contribute

You can suggest changes to the content of this repository through Git pull requests:
- If you are not among the repository owners, fork this repository
- Clone your fork locally
- Create a feature branch for your changes based on the `master` branch

### Pull Request Process

- Provide context for your changes in the pull request description
- Link related issues, if relevant
- Ask for merging into the `master` branch
- Require a review from one or several of the repository contributors

### For Repository Owners: Merging Process

- After the pull request has been approved by at least one other contributor, you can merge the pull request
- Prefer the "Squash and merge" option to merge multiple commits in a single one
